# Bakgrund

Ett litet företag i restaurangbranschen kontaktar dig för att utveckla ett litet personalregister.

## Krav
Programmet ska:

- Ta emot och lagra anställda med namn och lön via konsolen (ingen persistent lagring)
- Skriva ut registret i konsolen
